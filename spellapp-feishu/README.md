# SpellSnap 拼写助手 - 飞书小程序

一个帮你攻克英语拼写的飞书小程序。拍照录入单词，按记忆曲线复习，记录错误模式。

---

## 功能

| 页面 | 功能 |
|------|------|
| **录入** | 拍照识别单词（OCR）或手动输入，自动保存到词库 |
| **练习** | 按中文/读音提示拼写，即时反馈错误类型 |
| **词库** | 查看所有单词，显示复习状态，支持搜索和删除 |
| **分析** | 统计错误类型分布，导出数据给 AI 深度分析 |

---

## 核心机制

### Layer 1 错误分析（本地规则）
已内置 6 种常见拼写错误模式：
- 双写辅音遗漏（necessary → necesary）
- ie/ei 混淆（receive → recieve）
- 不发音 e 遗漏（love → lov）
- -tion/-sion 混淆
- -ant/-ent 混淆
- -able/-ible 混淆

### SM-2 记忆曲线
- 根据你的掌握程度（忘了/困难/记得/简单）调整下次复习时间
- 新单词：1天 → 3天 → 递增间隔
- 错了：回到1天后复习

## 如何运行

### 1. 安装飞书开发者工具
https://open.feishu.cn/document/client-sdk/devtool/overview

### 2. 创建飞书小程序
1. 登录 https://open.feishu.cn/app
2. 创建小程序 → 类型选「小程序」
3. 获取 App ID

### 3. 导入项目
1. 打开飞书开发者工具
2. 选择「导入项目」
3. 选择 `spellapp-feishu` 文件夹
4. 填入你的 App ID

### 4. 开发调试
- 点击「预览」可以在飞书 App 里测试
- 本地数据存在手机 Storage 里

## 待接入的 API（按优先级）

### P0 - 核心功能
- [ ] **OCR 文字识别**：拍照识别单词
  - 推荐：百度/腾讯/阿里 OCR API
  - 或者先用飞书自带的文档识别能力

- [ ] **词典 API**：自动填充释义和音标
  - 推荐：有道词典 API（免费额度够用）
  - 备选：爱词霸、海词

- [ ] **TTS 发音**：点击播放单词读音
  - 推荐：有道 TTS（免费）
  - 格式：`https://dict.youdao.com/dictvoice?audio={word}&type=2`

### P1 - 增强体验
- [ ] **云端同步**：数据备份到服务器
  - 可以用飞书多维表格当数据库
  - 或者用 LeanCloud / Firebase

- [ ] **AI 错误分析**：把错误记录发给 GPT 分析
  - 已实现导出功能，复制后手动发给 AI
  - 可选：接入 GPT API 自动分析

## 数据导出格式

### 给 AI 分析的错误记录
```json
[
  {
    "单词": "necessary",
    "释义": "必要的",
    "你的拼写": "necesary",
    "错误类型": "drop_double_consonant"
  }
]
```

复制到 Claude / GPT，提示词：
```
这是我的英语拼写错误记录，帮我分析错误规律，并按弱点类型归类...
（详见 stats.js 中的 copyForAI 方法）
```
